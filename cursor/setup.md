# Cursor Setup Instructions

## Option 1: Using .cursorrules (Recommended)

1. Copy the entire `cursor/` folder to your project root
2. Rename `.cursorrules` file if needed (Cursor auto-detects this filename)
3. Restart Cursor to reload rules

## Option 2: Using Cursor Rules File

1. In Cursor, open `Cmd/Ctrl + ,` (Settings)
2. Navigate to "Rules" section
3. Copy content from `cursor/.cursorrules`
4. Paste into your Cursor rules

## What's Included

- `.cursorrules` — Main AI rules for the Smart Store Pro plugin
- Includes: project context, build commands, coding rules, patterns, troubleshooting
