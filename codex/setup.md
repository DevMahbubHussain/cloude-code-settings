# Codex/Copilot Setup Instructions

## Option 1: Copilot Instructions File (Recommended)

1. Copy `.copilot-instructions.md` to your project root
2. Rename to `.copilot-instructions.md` (if not already)
3. Restart VS Code / GitHub Copilot

## Option 2: Using Copilot Chat

When using Copilot Chat, paste the contents of `.copilot-instructions.md` as context for each session.
